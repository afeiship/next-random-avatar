# next-random-avatar
> Random avatar api.

## install:
```bash
npm install -S afeiship/next-random-avatar --registry=https://registry.npm.taobao.org
```

## usage:
```js
import NxRandomAvatar from 'next-random-avatar';

NxRandomAvatar.men(/*1~100*/);
NxRandomAvatar.women(/*1~100*/);
NxRandomAvatar.lego(/*1~10*/);
```
